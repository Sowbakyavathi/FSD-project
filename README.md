# FSD Project

## Overview
This is a **full-stack application** for managing agricultural products like vegetables, fruits, and grains.  
It provides a **React frontend** for user interaction and a **Node.js backend** with a MongoDB database to store product details.

## Features
- Add, view, update, and delete products
- Responsive and interactive React frontend
- REST API backend with Node.js and Express
- MongoDB database integration
- User-friendly UI for managing products

## Tech Stack
- **Frontend:** React, HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB

## Installation & Setup

### Backend
1. Clone the repository:  
```bash
git clone https://github.com/Sowbakyavathi/FSD-project.git

Navigate to the backend folder:

cd FSD-project/backend

Install dependencies:

npm install

Start the backend server:

npm start

Frontend Setup

Navigate to the frontend folder:

cd ../frontend


Install dependencies:

npm install


Start the React app:

npm start


Open http://localhost:3000
 in your browser to view the app.

Usage

Use the frontend interface to manage products:

Add new products

View all products

Update existing products

Delete products

Example product (backend JSON):

{
  "name": "Tomato",
  "price": 30,
  "category": "Vegetables"
}

API Endpoints
Method	Endpoint	Description
GET	/products	Get all products
POST	/products	Add a new product
PUT	/products/:id	Update a product by ID
DELETE	/products/:id	Delete a product by ID
